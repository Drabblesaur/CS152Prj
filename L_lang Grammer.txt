L Language Grammer 

PROGRAM -> FUNCTION

FUNCTION -> FUNCTION IDENTIFIER (DECLATATIONS) BEGINLOCALS DECLATIONS ENDLOCAL BEGINBODY STATEMENT ENDBODY 

DELCARATIONS -> DELCATIONS DELCATION | EPSILON 
DELCARTION -> INDENTIFER: INT[NUMBER] | IDENTIFER: INT 

IDENTIFERS -> IDENTIFER | IDENTIFER, IDENTIFERS 

STATEMENTS -> STATEMENTS STATMENTS | STATEMENT SEMICOLON 
STATEMENT -> VAR | WHILE | DO | IF | ELSE IF | PRINT | PRINTLN 

WHILE -> WHILE       BEGINLOOP STATEMENTS ENDLOOP
DO -> DO  BEGINLOOP STATEMENTS ENDLOOP WHILE 
PRINT -> PRINT VAR 




VAR -> IDENTIFERS | INDENTIFER[EXPRESSION]
VARS -> VAR | VARS
